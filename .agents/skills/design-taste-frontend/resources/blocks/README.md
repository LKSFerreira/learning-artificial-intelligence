# Block Library

Contrato e schema: ver seção 12 de `../anti-slop-frontend-guide.md`.

Blocos (um arquivo por padrão) serão adicionados de forma iterativa, por exemplo:

```text
blocks/
  hero/
  feature/
  social-proof/
  pricing/
  cta/
  footer/
  navigation/
  portfolio/
  transition/
```

Ainda não há blocos implementados — use o guia + implementação sob demanda.
